# Scientific Paper Assistant with RAG and Evaluation

A reproducible Retrieval-Augmented Generation pipeline for scientific papers.

## Motivation

This project focuses on building a reliable and evaluable RAG system for scientific documents, with emphasis on:
- traceable retrieval
- source attribution
- evaluation

## Planned Features

- PDF ingestion
- Chunking
- Embeddings
- Retrieval
- Grounded answers
- Evaluation
